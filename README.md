# AltruisticXAI

> **Reflect. Decide. Grow.**

An ethical reflection engine for values-driven decision making. AltruisticXAI provides a calm, psychological "safe space" for reflection, guiding users through binary choices and helping build ethical clarity and self-awareness—all with radical privacy by default.

## 🌟 What is AltruisticXAI?

AltruisticXAI fills the modern gap in AI-driven ethical support—a "reflection engine" helping users:
- Clarify core values
- Make decisions with confidence and privacy
- Foster authentic growth through structured reflection

### Mission
Empower humans and teams to reflect, decide, and grow—with transparency and trust—through AI-facilitated, values-driven conversations that illuminate the ethical dimensions of every dilemma.

---

## ✨ Key Features

### 🎯 Ethical Reflection Layer
Conversational, values-first interface for clarity and trust—not just answers.

### 🔍 Transparent AI Reasoning
Every outcome includes "why"—rationale and values affected.

### 🔒 Radical Privacy
- User-owned data
- Private reflections with easy controls
- Local or cloud processing modes
- Export and delete controls

### 📊 Ripple Analysis
Understand the potential ripples of your decisions:
- Values upheld
- Values strained
- Long-term implications

### 📖 Reflection Memory
Private logbook/timeline of your reflections with easy search and export.

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/altruisticgenx/projectz.git
cd projectz
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### Build for Production

```bash
npm run build
```

This builds the app for production to the `build` folder.

## 🎨 Design System

### Color Palette
- **Pastel Neutrals**: Background and cards (#F5F5F7, #FAFAFA)
- **Ocean Blue**: Trust and calm (#4A90E2)
- **Sage Green**: Balance (#7ABEA8)
- **Deep Purple**: Primary CTA (#6B5B95)
- **Light Purple**: Hover states (#9B88B5)

### Typography
- **Primary**: Inter (body text)
- **Display**: Poppins (headers)

### Component Style
- Border radius: 16px (cards), 12px (buttons)
- Shadows: Subtle, layered
- Animations: Fade and slide, 300ms duration

For complete design specifications, see [UX_BLUEPRINT.md](./UX_BLUEPRINT.md).
